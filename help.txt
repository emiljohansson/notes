Available methods:

- add(note)     adds a note to content.txt
- remove(index) remove a note at index, use ls to find index
- ls()          list all notes
- help()        this information

Usage:

add/a:
node notes add The note you want to add.

remove/rm/r:
node notes remove 1

list/ls:
node notes ls

help/h:
node notes help
