Available methods:

- add(note)     adds a note to content.txt
- remove(index) remove a note at index, use list to find index
- update(index) update a note at index, use list to find index
- list()        list all notes
- help()        this information

Usage:

add/a:
notes-cli add The note you want to add.

remove/rm/r:
notes-cli remove 1

update/u:
notes-cli update 1 Update note at position 1

list/ls:
notes-cli list

help/h:
notes-cli help
